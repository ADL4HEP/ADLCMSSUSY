# ADLCMSSUSY
ADLs for CMS Run 2 SUSY analyses

This repository contains ADL descriptions of various published Run 2 CMS SUSY analyses, starting with those contributing to the pMSSM interpretation.

Definitions would be done either considering the original input datatier used by the analysis or adapted to Delphes for public use.

For more information on ADL: cern.ch/adl
